__construct()

TreeMap.php : 176

Constructs a TreeNode instance.

protected __construct(K|null $key, V|null $value, TreeNode<K, V>|null $NULL_GUARD) : mixed

If $NULL_GUARD is provided, initializes the node with the given key and value, and sets parent, left, and right to the sentinel node ($NULL_GUARD). If $NULL_GUARD is null, creates a sentinel node pointing to itself with no entry and height -1.

Parameters
$key : K|null

The key for the node's entry, or null for sentinel.

$value : V|null

The value for the node's entry, or null for sentinel.

$NULL_GUARD : TreeNode<K, V>|null

Sentinel node reference, or null to create sentinel.
